New Construction

  1. Open the downloaded eclipse.exe software.
  2. Click [File] -> [New] -> [Project]-> [C/C++] -> [C Project]
  3. Output project name Project name [hello word]
  4. In the Project type option, select [los Project]
  5. Click Flash or continue configuration

Compiling the code

  • Right click on the project -> [Build Project]
  • The compilation results are as follows: Dubeg directory generation [hello word.los] file
  • Code size: xxx               Txt size: xxx        Globel ram size: xxx   Init globel ram size: xxx              File size: xxx

Running los

  1. Click the icon button to run the los file. (In this project)

note

  1. The CDT plugin must be installed before operating the steps in this chapter.

los development use ide

  1. It is based on eclipse development.
  2. This article only describes the features developed specifically for los.

eclispe's toolbar

  • The icon button is to get the los device file.
  • Icon button is a package or package file system that will compile the project.
  • The icon button is the usb device that downloads the compiled file to los. (All files are re-downloaded).
  • The icon button is only for downloading los files. The * icon button is a download los file and a file that does not exist on the los device.
  • The icon button is the los file running on the pc side.
  • The icon button converts the current project's los file into an array file. (After the first conversion is completed, please right click on the Dubeg folder to refresh).
  • The icon button registers the current project on the pc.
  • The icon button is to open losshell in the los path of the current project.
  • icon button is set los.

losshell

  1. losshell is a tool for managing los.
  2. Please enter the ‘h’ or ‘help’ command before using.
  3. Note The eclipse TM plugin must be installed before using losshell.

Creating a new lgui interface file

  1. Click [File] -> [New] -> [Other] -> [lgui file] -> [Next]
  2. Output file name, and lcd size parameters, click Flash
  3. Right click on the newly created file, [Open With] -> [lgui editor]
  4. Click on the bottom of the pop-up window [lgui desige]

Open the file of the lgui interface

  1. Right click on the selected file [Open With] -> [lgui editor]
